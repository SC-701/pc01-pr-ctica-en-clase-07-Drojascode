# Autorizacion.Abstracciones

Interfaces y modelos compartidos del paquete de autorización por claims.
Dependencia interna — consumir a través de `Autorizacion.Middleware`.

- `ISeguridadDA` — contrato para el acceso a datos de seguridad
- `IAutorizacionFlujo` — contrato para la lógica de claims
- `Perfil` — modelo que representa un rol de usuario
